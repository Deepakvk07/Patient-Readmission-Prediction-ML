# Hospital Readmission Predictor

## Overview
This project predicts whether a discharged patient is likely to be readmitted within 30 days using healthcare and clinical data.

The model analyses patient medical history, diagnoses, demographics, medications, and hospital-related features to identify readmission risk.

This is a real-world healthcare machine learning use case for predictive analytics and risk scoring.

## Features

- Feature encoding
- Exploratory Data Analysis (EDA)
